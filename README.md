# Machine Learning Course Project

Course experiments and report materials for a machine learning class.

## Overview

This repository contains several experiment modules covering classic machine learning methods and deep learning models, together with notebooks, source code, and selected results.

Current modules include:

- `module_a_xor` — BP neural network for XOR classification
- `module_b_svm` — SVM for binary and multi-class classification
- `module_c_digits` — PCA, K-means, and GMM/EM on handwritten digits
- `module_g_cnn` — CNN for handwritten digit recognition
- `module_h_vae` — Convolutional VAE for representation learning and generation

## Repository Structure

Each module is organized in a similar way:

- `src/` — core implementation
- `notebooks/` — experiment notebooks
- `results/` — figures, tables, and logs

## Main Topics

- Backpropagation on XOR
- Support Vector Machines
- PCA for dimensionality reduction
- K-means clustering
- EM for Gaussian Mixture Models
- CNN-based image classification
- VAE-based representation learning

## Notes

- The project emphasizes understanding algorithm principles, experiment design, visualization, and result analysis.
- Some experiments aim to reduce reliance on highly encapsulated algorithm APIs and keep the implementation as explicit as possible.

## Report

This repository is used together with the course report.  
Code organization and experiment outputs here are intended to support reproducibility and documentation.
